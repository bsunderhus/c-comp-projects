Instalação:
- para instalar clojure em um  sistema linux digite: sudo apt-get install leiningen
(leiningen é um gerenciador de projetos que ao instala-lo o clojure também é instalado).

Run:
- para rodar o programa basta estar na pasta /trab3, adicionar o arquivo entrada.txt e
rodar o comando lein run *argumento* (ex: lein run 3, criando 3 grupos).
o programa vai gerar na pasta /trab3 o saida.txt e o result.txt.
